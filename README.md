# Trabajo Práctico Integrador - CI/CD

Proyecto Python para el procesamiento de compras de un supermercado.

## Objetivo

Implementar un flujo de trabajo con GitHub, ramas, Pull Requests, pruebas automatizadas y GitHub Actions.

## Archivos principales

- `tp.py`: código base del sistema.
- `requirements.txt`: dependencias del proyecto.
- `.gitignore`: archivos que no deben subirse al repositorio.

## Ejecución

```bash
python tp.py
